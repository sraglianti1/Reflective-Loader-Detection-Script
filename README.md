Lo script è parte/risultato del lavoro di tesi triennale in Informatica. La presentazione contiene una rapida panoramica del lavoro svolto, contestualizzandolo e spiegandolo brevemente. Lo script è scritto in Python e provvede a rilevare in maniera automatica il Reflective Loader del ransomware WannaCry. Uno degli obiettivi dello script era di poter essere impiegato con successo anche su casi che adottassero strategie similari a quelle di WannaCry. Lo sviluppo è puramente sperimentale e perciò non ha particolari pretese di successo o efficienza, bensì solamente teoriche e dimostrative.
